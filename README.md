redis-server.exe redis.windows.conf

这时候另启一个 cmd 窗口，原来的不要关闭，不然就无法访问服务端了。切换到 redis 目录下运行:
redis-cli.exe -h 127.0.0.1 -p 6379

set myKey abc

取出键值对:
get myKey

获取所有的 key：
keys \*

删除当前数据库中的所有 Key
